Example 01 Print debugging to the screen using print()

example_01()
This debug message uses print.

The next example uses logging instead of print(). Note that this is almost as simple as print(). In the following examples, we’ll see why logging is better than print!

example_02

 example_02 ()

Example 02 Log to the screen with the root logger.

example_02()
"This debug message uses the root logger."

We can get the root logger from anywhere in our application with logging.getLogger() We already configured the root logger in example_02, but we can modify its behavior here. This illustrates two handy things about logging:

  • logging provides access to the loggers from anywhere, so the developer does not have to search the code.
  • We can change the level of messages to suppress debug messages.
